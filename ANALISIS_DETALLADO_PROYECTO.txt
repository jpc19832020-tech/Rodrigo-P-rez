1. DESCRIPCIÓN GENERAL DEL PROYECTO
===========================================

Este proyecto es una tarjeta digital profesional (vCard) desarrollada para Rodrigo Pérez, CEO de EXO Digital Studio. Se trata de una aplicación web moderna y responsive que funciona como una tarjeta de presentación digital interactiva con múltiples funcionalidades de contacto y compartición.

El proyecto utiliza una arquitectura basada en configuración centralizada, lo que permite fácil personalización y mantenimiento. Está diseñado con un enfoque mobile-first y optimizado para SEO y redes sociales.

4. ANÁLISIS DETALLADO DE COMPONENTES
===========================================

4.1. ARCHIVO CONFIG.JS (121 líneas)
------------------------------------
Este es el corazón del sistema de configuración centralizada. Contiene:

SECCIONES PRINCIPALES:
- personalInfo: Datos personales (nombre, título, subtítulo, foto)
- contactInfo: Información de contacto (teléfono, email, redes sociales)
- companyInfo: Información de EXO Digital Studio
- urls: URLs y enlaces importantes
- seo: Metadatos para optimización SEO
- images: Rutas de todas las imágenes del proyecto
- uiTexts: Textos de la interfaz en español
- brandColors: Paleta de colores de la marca
- messages: Mensajes predefinidos para WhatsApp y email
- config: Configuración adicional (año actual, copyright, etc.)

VENTAJAS DE ESTA ARQUITECTURA:
- Centralización de toda la información personalizable
- Fácil mantenimiento sin necesidad de modificar código
- Reutilización para diferentes personas/empresas
- Prevención de errores de sincronización

4.2. ARCHIVO INDEX.HTML (323 líneas)
------------------------------------
Estructura semántica moderna con las siguientes secciones:

METADATOS Y SEO:
- Meta tags charset, viewport, title, description
- Open Graph para Facebook/LinkedIn
- Twitter Cards para Twitter
- Favicon y apple-touch-icon

ESTRUCTURA PRINCIPAL:
- Botón de cambio de idioma (ES/EN)

- Sección de contacto con múltiples botones
- Botones de compartir (tarjeta y QR)

FUNCIONALIDADES INTERACTIVAS:

4.3. ARCHIVO MAIN.JS (941 líneas)
